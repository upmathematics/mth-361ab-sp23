Difference of Two Proportions Relevant Variables Two two-level categorical
Population Parameters \(p_a \longrightarrow\) population proportion for group a

\(p_b \longrightarrow\) population proportion for group b

\(p_0 \longrightarrow\) null value
Sample Statistics \(\hat{p}_a \longrightarrow\) sample proportion (point estimate) for group a

\(\hat{p}_b \longrightarrow\) sample proportion (point estimate) for group b

\(n_a \longrightarrow\) sample size for group a

\(n_b \longrightarrow\) sample size for group b
Theoretical Conditions Independent random samples

Success-failure:
\(\hspace{30px} n_a \hat{p}_a \ge 10\) and \(n_b\left(1-\hat{p}_b\right) \ge 10\) for group a
\(\hspace{30px} n_b \hat{p}_b \ge 10\) and \(n_b\left(1-\hat{p}_b\right) \ge 10\) for group b
Hypothesis Testing \(H_0: p_a = p_b \text{ or } p_a - p_b = 0 \longrightarrow\) null hypothesis

\(H_A: \begin{cases} p_a \ne p_b \text{ or } p_a - p_b \ne 0 \\ p_a > p_b \text{ or } p_a - p_b > 0 \\ p_a < p_b \text{ or } p_a - p_b < 0 \end{cases} \longrightarrow\) alternative hypothesis

\(\alpha \longrightarrow\) significance level (type I error rate)

\(\hat{p}_{pool} = \frac{\hat{p}_a n_a + \hat{p}_bn_b}{n_a + n_b} \longrightarrow\) pooled sample proportion

\(SE_{H} = \sqrt{\hat{p}_{pool}\left(1-\hat{p}_{pool}\right) \left( \frac{1}{n_a} + \frac{1}{n_b} \right)} \longrightarrow\) standard error

\(z = \frac{\left(\hat{p}_a - \hat{p}_b \right) - p_0}{SE_{H}} \longrightarrow\) test statistic (z-score)

\(\text{p-value} = \begin{cases} P(Z < -z) \text{ or } P(Z > z) & \rightarrow \text{ one-sided} \\ 2P(Z < -z) \text{ or } 2P(Z > z) & \rightarrow \text{ two-sided} \end{cases}\) 		\(P(Z < -z) =\) pnorm(-z,0,1)

\(P(Z > z) =\) 1-pnorm(z,0,1)

\(P(Z = z) =\) dnorm(z,0,1)

\(ME = z^* SE_{CI} \longrightarrow\) margin of error
