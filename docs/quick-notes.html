One Proportion Relevant Variables Response \(\longrightarrow\) One two-level categorical
Population Parameters \(\displaystyle p \longrightarrow\) population proportion

\(\displaystyle p_0 \longrightarrow\) null value
Sample Statistics \(\displaystyle \hat{p} \longrightarrow\) population proportion (point estimate)

\(\displaystyle n \longrightarrow\) sample size
Theoretical Conditions Independent random samples

Success-failure:
\(\hspace{30px} np \ge 10\) and \(n(1-p) \ge 10\)
Hypothesis Testing \(\displaystyle H_0: p = p_0 \longrightarrow\) null hypothesis

\(\displaystyle H_A: p \ge p_0, p \le p_0, \text{ or } p \ne p_0 \longrightarrow\) alternative hypothesis

\(\displaystyle \alpha \longrightarrow\) significance level (type I error rate)

\(\displaystyle SE_{H} = \sqrt{\frac{p_0\left(1-p_0\right)}{n}} \longrightarrow\) standard error

\(\displaystyle z = \frac{\hat{p}-p_0}{SE_{H}} \longrightarrow\) test statistic (z-score)

\(\displaystyle \text{p-value} = \begin{cases} P(Z \le -z) \text{ or } P(Z \ge z) &amp; \rightarrow \text{ one-sided} \\ 2P(Z \le -z) \text{ or } 2P(Z \ge z) &amp; \rightarrow \text{ two-sided} \end{cases}\) 		\(P(Z \le -z) =\) pnorm(-z,0,1)

\(P(Z \ge z) =\) 1-pnorm(z,0,1)

\(P(Z = z) =\) dnorm(z,0,1)
Confidence Interval \(\displaystyle SE_{CI} = \sqrt{\frac{\hat{p}\left(1-\hat{p}\right)}{n}} \longrightarrow\) standard error

\(\displaystyle ME = z^* SE_{CI} \longrightarrow\) margin of error

\(\displaystyle cl = 1-\alpha \longrightarrow\) confidence level

\(\displaystyle z^* \longrightarrow\) critical value at \(cl\) confidence level

\(\displaystyle \hat{p} \pm ME \longrightarrow\) lowerbound and upperbound of the interval 		alpha \(= \alpha\)

cl \(= 1-\alpha\)

\(z^* =\) qnorm(cl + (alpha/2),0,1)

\(-z^* =\) qnorm(cl - (alpha/2),0,1)
Difference of Two Proportions Relevant Variables Response \(\longrightarrow\) Two two-level categorical
Population Parameters \(\displaystyle p_a \longrightarrow\) population proportion for group a

\(\displaystyle p_b \longrightarrow\) population proportion for group b

\(\displaystyle p_0 \longrightarrow\) null value
Sample Statistics \(\displaystyle \hat{p}_a \longrightarrow\) sample proportion (point estimate) for group a

\(\displaystyle \hat{p}_b \longrightarrow\) sample proportion (point estimate) for group b

\(\displaystyle n_a \longrightarrow\) sample size for group a

\(\displaystyle n_b \longrightarrow\) sample size for group b
Theoretical Conditions Independent random samples

Success-failure:
\(\hspace{30px} n_a \hat{p}_a \ge 10\) and \(n_b\left(1-\hat{p}_b\right) \ge 10\) for group a
\(\hspace{30px} n_b \hat{p}_b \ge 10\) and \(n_b\left(1-\hat{p}_b\right) \ge 10\) for group b
Hypothesis Testing \(\displaystyle H_0: p_a = p_b \text{ or } p_a - p_b = 0 \longrightarrow\) null hypothesis

\(\displaystyle H_A: \begin{cases} p_a \ne p_b \text{ or } p_a - p_b \ne 0 \\ p_a \ge p_b \text{ or } p_a - p_b \ge 0 \\ p_a \le p_b \text{ or } p_a - p_b \le 0 \end{cases} \longrightarrow\) alternative hypothesis

\(\displaystyle \alpha \longrightarrow\) significance level (type I error rate)

\(\displaystyle \hat{p}_{pool} = \frac{\hat{p}_a n_a + \hat{p}_bn_b}{n_a + n_b} \longrightarrow\) pooled sample proportion

\(\displaystyle SE_{H} = \sqrt{\hat{p}_{pool}\left(1-\hat{p}_{pool}\right) \left( \frac{1}{n_a} + \frac{1}{n_b} \right)} \longrightarrow\) standard error

\(\displaystyle z = \frac{\left(\hat{p}_a - \hat{p}_b \right) - p_0}{SE_{H}} \longrightarrow\) test statistic (z-score)

\(\displaystyle \text{p-value} = \begin{cases} P(Z \le -z) \text{ or } P(Z \ge z) &amp; \rightarrow \text{ one-sided} \\ 2P(Z \le -z) \text{ or } 2P(Z \ge z) &amp; \rightarrow \text{ two-sided} \end{cases}\) 		\(P(Z \le -z) =\) pnorm(-z,0,1)

\(P(Z \ge z) =\) 1-pnorm(z,0,1)

\(P(Z = z) =\) dnorm(z,0,1)
Confidence Interval \(\displaystyle SE_{CI} = \sqrt{\frac{\hat{p}_a\left(1-\hat{p}_a\right)}{n_a} + \frac{\hat{p}_b\left(1-\hat{p}_b\right)}{n_b}} \longrightarrow\) standard error

\(\displaystyle ME = z^* SE_{CI} \longrightarrow\) margin of error

\(\displaystyle cl = 1-\alpha \longrightarrow\) confidence level

\(\displaystyle z^* \longrightarrow\) critical value at \(cl\) confidence level

\(\displaystyle \left(\hat{p}_a - \hat{p}_b\right) \pm ME \longrightarrow\) lowerbound and upperbound of the interval 		alpha \(= \alpha\)

cl \(= 1-\alpha\)

\(z^* =\) qnorm(cl + (alpha/2),0,1)

\(-z^* =\) qnorm(cl - (alpha/2),0,1)

Two-Way Tables Relevant Variables Explanatory \(\longrightarrow\) One categorical with two or more levels
Response \(\longrightarrow\) One categorical with two or more levels
Population Parameters \(\displaystyle df \longrightarrow\) degrees of freedom (expected value of the chi-square distribution)
Sample Statistics \(\displaystyle \chi^2 \longrightarrow\) chi-squared statistic (point estimate)

\(\displaystyle C \longrightarrow\) number of levels in variable A (number of table columns)

\(\displaystyle R \longrightarrow\) number of levels in variable B (number of table rows)

\(\displaystyle n \longrightarrow\) sample size
Theoretical Conditions Independent random samples

Sample size for each cell in the two-way table must have at least 5 expected cases
Hypothesis Testing \(\displaystyle H_0:\) Variable A is independent with variable B \(\longrightarrow\) null hypothesis

\(\displaystyle H_A:\) Variable A is not independent with variable B \(\longrightarrow\) alternative hypothesis

\(\displaystyle \alpha \longrightarrow\) significance level (type I error rate)

\(\displaystyle df = (C-1)(R-1) \longrightarrow\) degrees of freedom

\(\displaystyle E_k = E_{(i,j)} = \frac{total(row(i)) \times total(column(j))}{n} \longrightarrow\) expected value of cell \(k\) for row \(i\) and column \(j\)

\(\displaystyle O_k \longrightarrow\) number of observation of cell \(k\)

\(\displaystyle T = C*R \longrightarrow\) number of table cells

\(\displaystyle \chi^2 = \sum_{k=1}^T \frac{(O_k - E_k)^2}{E_k} \longrightarrow\) test statistic (chi-squared)

\(\displaystyle \text{p-value} = P(X \ge \chi^2) = 1 - P(X \le \chi^2) \longrightarrow\) where \(X \sim \chi^2\) 		chisq_stat \(= \chi^2\)

df \(= (C-1)(R-1)\)

\(P(X \le \chi^2) =\) pchisq(chisq_stat,df)

\(P(X \ge \chi^2) =\) 1 - pchisq(chisq_stat,df)
